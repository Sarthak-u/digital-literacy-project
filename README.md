## 📌 Project Overview
This project was completed as part of the Digital Literacy course. The aim of this project is to develop awareness about 
digital tools, online safety, professional communication, and cybercrime prevention. Through this project, I explored 
various platforms and created resources that can help students become responsible digital citizens.
